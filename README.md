# Elementor Link Shortener Button Widget

این افزونه یک ویجت دکمه به المنتور اضافه می‌کند و **کاملاً آفلاین** روی هاست خودتان کار می‌کند.

## رفتار ویجت
- کاربر روی دکمه کلیک می‌کند.
- لینک همان صفحه با AJAX به سرور ارسال می‌شود.
- افزونه داخل وردپرس یک کد کوتاه می‌سازد و در دیتابیس سایت نگه می‌دارد.
- لینک کوتاه داخلی (مثل `/?elsb_go=abc1234`) ساخته می‌شود.
- لینک کوتاه به‌صورت خودکار در کلیپ‌بورد کپی می‌شود.

## نکته مهم
- این افزونه به هیچ سرویس خارجی (مثل is.gd یا Bitly) وصل نمی‌شود.
- بنابراین حتی روی هاست بدون دسترسی بیرونی هم کار می‌کند.

## نصب
1. پوشه افزونه را در `wp-content/plugins` قرار دهید.
2. افزونه را از پنل وردپرس فعال کنید.
3. در المنتور، ویجت `دکمه کوتاه‌کننده لینک صفحه` را جست‌وجو و به صفحه اضافه کنید.
